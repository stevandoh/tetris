click on the appropriate link based on your OS

##### Installation######

1. Windows
Go to the folder named windows_version
If you re running the game for the first time 
	* if your python version is 32 bits double click on setup32.bat
	* if your python version is 64 bits double click on setup64.bat
Then double click on runTetris.bat to play the game 

2. Unix based os 
Open you terminal 
go to the location the unixVersion folder 
enter python3 tetris.py
